Mission

The DocCookBook project aims to create a collection of open source, professional solutions in a cookbook-style for the international DocBook community.

Introduction

This is the home of the DocCookBook, a free book about DocBook. It is written in a cookbook-style which gives you lots of solutions to your problems.

Currently, there is only a HTML version available. Other formats are planned, but depends on your feedback.

Feedback/Contributing

There are lots of tasks where you can help:

  • Give feedback to existing topics
  • Send ideas for new topics
  • Open a ticket, if you think there is something wrong
  • Send patches for existing topics
  • Write completely new topics.
  • Improve the look-and-feel of the HTML page

Translations

At the moment, the book is work in progress. Things will change, new topics appear and old are removed, revised, or moved to a different chapter. For this reason, I would first like to create a beta version which is stable enough for translation and do not introduce too much pain.
