Mission

The DocCookBook project aims to create an open source book about DocBook and the DocBook XSL stylesheets written in a cookbook-style manner and released under a Creative Commons license.

Reading The Book

Currently, there is only HTML available. Other formats will be provided in the near future.

The Exo Typeface Family

Another project you may find interesting is the Exo typeface family to create a free geometric sans font family. Natanael designed the Exo typeface family with some really unique features: 9 weights, upright and italic style, 700 characters, ligatures, small caps, old style figures, and many other OpenType features.

More about these magnificant fonts can be found on the Exo project page. Please help Natanael to fund his project and to create an open source font for everyone.
